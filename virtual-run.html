VIRTUAL RUN PLATFORM

จัดงาน Virtual Run ได้ง่ายๆ ครบจบในแพลตฟอร์มเดียว

ระบบที่ออกแบบมาเพื่อผู้จัดงานวิ่ง Virtual Run โดยเฉพาะ ครอบคลุมทุกขั้นตอนตั้งแต่เปิดรับสมัครจนถึงจัดส่ง Race Pack ให้นักวิ่ง

✓

ระบบสมัครออนไลน์

หน้าสมัครสวยงาม ชำระเงินได้หลายช่องทาง รองรับนักวิ่งหลายหมื่นคน

✓

ส่งผลผ่านแอป

นักวิ่งส่งผลได้ง่ายผ่าน GPS Tracking หรืออัพโหลดภาพหน้าจอจากแอปวิ่ง

✓

จัดส่ง Race Pack ถึงบ้าน

ระบบจัดการคลังสินค้าและขนส่ง ส่ง Race Pack ถึงมือนักวิ่งทั่วประเทศ

✓

E-BIB & E-Certificate

ระบบสร้างบิบอิเล็กทรอนิกส์และใบประกาศนียบัตรอัตโนมัติ

✓

Dashboard สำหรับผู้จัดงาน

ดูยอดสมัคร รายได้ สถานะการจัดส่ง Real-time ได้ตลอดเวลา

พร้อมจัดงานวิ่งกับเรา?

ไม่ว่าจะเป็นงาน Virtual Run หรืองานวิ่งจริง เรามีโซลูชันครบวงจรสำหรับคุณ

ติดต่อเราวันนี้

Running Solution Platform ครบวงจร สำหรับผู้จัดงานวิ่งทั่วประเทศ
