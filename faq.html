คำถามที่พบบ่อย

มีคำถาม? เรามีคำตอบ

Checkrace ให้บริการครบวงจร ได้แก่ ระบบ Virtual Run Platform, ระบบจับเวลาด้วยชิป RFID, และรับผลิตอุปกรณ์การแข่งขัน เช่น เสื้อวิ่ง เหรียญรางวัล บิบ

ระบบจับเวลาของเรารองรับนักวิ่งได้มากกว่า 30,000 คนต่องาน พร้อมแสดงผลแบบ Real-time บนหน้าจอและแอปพลิเคชัน

ได้ครับ เรามีทีมออกแบบพร้อมรับผลิตเหรียญตามดีไซน์ที่ต้องการ รวมถึงเหรียญ Finisher Medal, Series Medal, และเหรียญพิเศษ

ค่าบริการขึ้นอยู่กับรูปแบบงานและจำนวนนักวิ่ง กรุณาติดต่อทีมงานเพื่อรับใบเสนอราคาเฉพาะงานของคุณ

นักวิ่งสมัครผ่านเว็บไซต์ Checkrace เลือกระยะทาง วิ่งตามเงื่อนไข ส่งผลผ่านแอป และรอรับ Race Pack ที่บ้าน ง่ายและสะดวก

พร้อมจัดงานวิ่งกับเรา?

ไม่ว่าจะเป็นงาน Virtual Run หรืองานวิ่งจริง เรามีโซลูชันครบวงจรสำหรับคุณ

ติดต่อเราวันนี้

Running Solution Platform ครบวงจร สำหรับผู้จัดงานวิ่งทั่วประเทศ
